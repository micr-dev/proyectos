# proyectos

Portfolio site for [micr.dev](https://micr.dev) — a curated collection of software, hardware, MCPs, and skills.

## Tech Stack

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Animation**: Framer Motion
- **Smooth Scroll**: Lenis

## Development

```bash
pnpm install
pnpm dev
```

## Structure

- `app/` — Next.js app router pages
- `components/` — Reusable UI components
- `public/` — Static assets

## Languages

TypeScript, CSS
